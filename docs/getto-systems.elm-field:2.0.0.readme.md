

## Contributes

### Test

fix codes, and pass tests

```
$ npm test
```


### Update

run `bin/bump.sh`, and create release-request

```
$ ./bin/bump.sh
$ git add elm.json
$ git release-request exact "RELEASE MESSAGE"
```
