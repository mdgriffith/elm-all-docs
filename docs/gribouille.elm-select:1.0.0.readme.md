# elm-select

ELM select component (in progress...).


## Example

```elm
view : Html msg
view =
  Select.view (Select.default pipe defaultItems) items
```


## Development

Execute the example:

```
> cd examples
> npm run dev
```

open `http://0.0.0.0:3000`.
